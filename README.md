# 🛒 Retail Sales Analysis & Prediction Dashboard

A real-world Data Science project built using Python, Streamlit, Pandas, Matplotlib, and Scikit-learn. This application performs complete retail sales data analysis, cleaning, visualization, and machine learning-based sales prediction through an interactive dashboard.

---

## 🚀 Features

- Upload retail sales CSV datasets
- Automatic data cleaning and preprocessing
- Handle missing values and duplicates
- Statistical analysis and summaries
- Interactive sales trend visualization
- Histogram and correlation heatmap generation
- Scatter plot analysis
- Machine Learning sales prediction using Linear Regression
- Actual vs Predicted comparison chart
- Download cleaned dataset
- Interactive Streamlit dashboard

---

## 📦 Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/retail-sales-prediction.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd retail-sales-prediction
```

### 3️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

OR manually install:

```bash
pip install streamlit pandas numpy matplotlib scikit-learn
```

---

## ▶️ Run the Application

```bash
streamlit run retail_sales_prediction.py
```
